# AI Sentiment Analyzer

## 📌 Description
Ce projet est un petit outil en Python qui analyse le sentiment des messages clients (positif, négatif, neutre) et génère des statistiques simples.

Il simule un système utilisé dans les centres de support client pour aider à prioriser les demandes.

---

## ⚙️ Fonctionnalités
- Analyse de sentiment basique (rule-based)
- Historique des messages
- Sauvegarde des données dans un fichier local
- Génération de statistiques simples
- Rapport global de performance

---

## 🧠 Objectif du projet
Améliorer la compréhension des messages clients et simuler un outil de support utilisé en entreprise.

---

## 🛠️ Technologies utilisées
- Python 3
- Programmation modulaire
- Manipulation de fichiers

---

python3 main.py
